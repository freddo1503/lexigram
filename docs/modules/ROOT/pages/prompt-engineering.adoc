= Prompt Engineering
:description: Agent prompt design in French, caption rules, image style guidelines.

Agent prompts are defined in `app/config/agents.yml`, written entirely in French.
Task guardrails are enforced in `app/agents/task.py`.

== Design Principles

* *Language:* All prompts in French -- matches the domain (French law) and output language
* *Persona-driven:* Each agent has a detailed backstory that shapes its behavior
* *Structured prompts:* XML-style tags (`<role>`, `<instructions>`, `<constraints>`) per Mistral prompting guide
* *Citizen-first:* Aligned with project vision — accessible, neutral, no advice
* *Anti-hallucination:* Caption task explicitly forbids inventing details absent from source
* *Guardrails:* Validation functions enforce output constraints programmatically

== Agent Personas

=== AnalysteJuridique

* *Role:* "Vulgarisateur Juridique Expert"
* *Backstory emphasis:* Simplify French law for the general public, translate legal jargon into plain language
* *Behavioral directive:* Citizen-first framing, adaptive lens by law type, no advice

=== LexPictor

* *Role:* "Directeur Artistique Numerique"
* *Backstory emphasis:* Create conceptual images for social media, translate abstract concepts visually
* *Behavioral directive:* Minimalist, modern, evocative — no text, logos, or national symbols

=== LexMarker

* *Role:* "Redacteur Instagram specialise en vulgarisation juridique"
* *Backstory emphasis:* Adapt legal content for Instagram, never invent details absent from source
* *Behavioral directive:* Faithful to source, emojis mandatory, conditional for uncertain facts

== Output Constraints

=== Text Summary

Required sections (enforced by `validate_text_summary`):

. "Titre officiel" -- official title of the legal text
. "Date de publication" -- in "day month year" format
. "Signataires officiels" -- official signatories
. Per-article analysis with number/identifier + content summary + implications

Minimum: 50 words.

=== Image Generation

Style rules (from agents.yml):

* Minimalist, modern, soft colors
* Legal/legislative/official context
* *Exclude:* text, logos, national flags

Validated by `validate_image_payload`: must produce `ImagePayload` with non-empty `image_url` (must contain S3 bucket name) and `image_description`.

=== Instagram Caption

Constraints (enforced by `validate_caption`):

[%header,cols="1,2"]
|===
|Rule |Value

|Max characters |2200 (Instagram limit)
|Min words |20
|Hashtags |5-10 required (max 30)
|Emojis |At least 1 required
|Formatting |No `*` or `_` (plain text only)
|===

Required structure (from agents.yml):

. Captivating hook summarizing essential points
. Key points in simple language
. Practical implications for citizens
. Publication date and signatories reference
. 5-10 relevant hashtags

== Guardrail Implementation

Guardrails are Python functions passed to CrewAI's `Task(guardrail=...)` parameter.
Each returns `Tuple[bool, str | TaskOutput]` -- `(valid, error_message_or_output)`.
Tasks have `guardrail_max_retries=3` -- if validation fails, CrewAI retries the task.

NOTE: The caption guardrail strips `*` and `_` characters instead of rejecting, since Mistral LLMs produce markdown formatting despite explicit instructions and Instagram renders plain text only.
