= System Overview
:description: What Lexigram is, high-level architecture, and tech stack.

Lexigram is an AI-powered multi-agent system that summarizes French legislation and publishes daily Instagram posts.
It runs as an AWS Lambda function triggered by EventBridge on weekdays at 11:30 UTC.

== What It Does

. Syncs new laws from the Legifrance API (official French law database)
. Picks the oldest unprocessed law
. Runs a 3-agent CrewAI pipeline: legal analysis -> image generation -> Instagram caption
. Publishes the result to Instagram with a comment linking back to the Legifrance source

== Architecture

[source]
----
EventBridge (cron: weekdays 11:30)
  -> Lambda (Docker, Python 3.12, 512MB, 4min timeout)
       -> Legifrance API (pylegifrance)          [law sync + fetch]
       -> DynamoDB                                [track processed laws]
       -> CrewAI Pipeline
            -> Mistral Large (text analysis)      [AnalysteJuridique]
            -> GPT-4 + DALL-E 3 (image)           [LexPictor]
            -> Mistral Large (caption)             [LexMarker]
       -> Instagram Graph API                     [publish post + comment]
       -> AWS Secrets Manager                     [API keys, tokens]
----
