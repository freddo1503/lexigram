= AI Agents
:description: Three CrewAI agents: AnalysteJuridique, LexPictor, LexMarker -- roles, goals, models, tools.

Lexigram uses three specialized CrewAI agents, each with a distinct LLM and role.
All agents load their role, goal, and backstory from `app/config/agents.yml` (written in French).

== Agent Summary

[%header,cols="1,1,1,1,1"]
|===
|Agent |Class |LLM |Tools |Delegation

|AnalysteJuridique |`app/agents/lex_analyst.py` |Mistral Large |None |No
|LexPictor |`app/agents/lex_pictor.py` |GPT-4 |DalleTool |No
|LexMarker |`app/agents/lex_marketer.py` |Mistral Large |None |No
|===

== AnalysteJuridique

Legal text analysis specialist.
Extracts key points from French legislation and produces structured analyses.

* *Model:* `mistral/mistral-large-latest` (API key: `settings.mistral_api_key`)
* *Role:* "Analyste Juridique Expert"
* *Goal:* Provide rigorous, synthetic, precise legal analyses with essential implications
* *Verbose:* True

== LexPictor

Visual artist agent that generates illustrative images from legal summaries.

* *Model:* `gpt-4` (API key: `settings.openai_api_key`)
* *Role:* "Artiste Visuel Innovant"
* *Goal:* Create captivating visual artworks evoking emotions and communicating complex concepts
* *Tools:* `[DalleTool()]` -- see xref:dalle-image-generation.adoc[DALL-E Image Generation]
* *Delegation:* Disabled
* *Verbose:* True

== LexMarker

Social media specialist that transforms legal analyses into Instagram-ready captions.

* *Model:* `mistral/mistral-large-latest` (API key: `settings.mistral_api_key`)
* *Role:* "Specialiste en adaptation et diffusion de contenu juridique sur Instagram"
* *Goal:* Convert legal analyses into impactful, accessible Instagram publications
* *Delegation:* Disabled
* *Verbose:* True

== Configuration

Agent personas are defined in `app/config/agents.yml` under the `agents` key.
Each agent constructor reads from `agents_config["agents"]["<agent_name>"]`.

The YAML keys per agent: `role`, `goal`, `backstory`.
All prompts are written in French to match the domain (French legislation) and output language.
