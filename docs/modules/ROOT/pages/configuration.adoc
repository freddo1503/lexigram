= Configuration
:description: LexigramSettings (Pydantic), SettingsManager, env vars, agents.yml loading.

Configuration is centralized in `app/config.py`.

== LexigramSettings

Pydantic `BaseSettings` with field validation.

[%header,cols="1,1,1,2"]
|===
|Field |Type |Default |Description

|`log_level` |str |`"INFO"` |Validated: DEBUG, INFO, WARNING, ERROR, CRITICAL
|`aws_secret_name` |str |`"my-env-secrets"` |AWS Secrets Manager secret name
|`legifrance_client_id` |Optional[str] |None |Legifrance API client ID
|`legifrance_client_secret` |Optional[str] |None |Legifrance API client secret
|`mistral_api_key` |Optional[str] |None |Mistral AI API key (all agents)
|`instagram_access_token` |Optional[str] |None |Instagram long-lived user access token
|`dynamo_table_name` |Optional[str] |None |DynamoDB table name
|`s3_bucket_name` |Optional[str] |None |S3 bucket name for generated images
|===

== SettingsManager

Singleton pattern managing settings + API clients.

* `settings` property: returns `LexigramSettings` instance
* `api_config` property: creates `pylegifrance.ApiConfig.from_env()` (reads credentials from environment variables)
* `api_client` property: creates `pylegifrance.LegifranceClient(api_config)`

=== Secret Loading

`_load_secrets()` flow:

. Calls `get_all_secrets()` which fetches from AWS Secrets Manager
. Updates `LexigramSettings` fields with secret values
. Sets `LEGIFRANCE_CLIENT_ID` and `LEGIFRANCE_CLIENT_SECRET` as environment variables (required by pylegifrance)

== agents.yml Loading

`load_agents_config()` reads `app/config/agents.yml` and returns a `dict[str, dict]`.

The config file path: `Path(__file__).parent / "config" / "agents.yml"`.

== Global Instances

Module-level singletons created at import time:

[source,python]
----
settings_manager = SettingsManager()
settings = settings_manager.settings
agents_config = load_agents_config()
----

These are imported throughout the codebase via `from app.config import settings, agents_config`.

=== Lazy API Accessors

Two module-level functions provide lazy access to the pylegifrance API client:

[source,python]
----
get_api_config()   # -> ApiConfig (via settings_manager.api_config)
get_api_client()   # -> LegifranceClient (via settings_manager.api_client)
----

Used by services like `LawSyncService` and `LawProcessingService` instead of importing the client directly.

== Environment Variable Defaults

At module load time, `app/config.py` sets two default environment variables:

* `OTEL_SDK_DISABLED=true` -- disables OpenTelemetry SDK (avoids instrumentation overhead)
* `CREWAI_STORAGE_DIR=/tmp` -- directs CrewAI storage to `/tmp` (required for Lambda's read-only filesystem)

== Local Development

Use a `.env` file at project root. `LexigramSettings` inherits from Pydantic `BaseSettings` which automatically reads `.env` files.
In production (Lambda), secrets come from AWS Secrets Manager.
